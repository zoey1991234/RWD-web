# RWD-web
 
開發模式：`npm run dev`

構建應用：`npm run build`

預覽生產版本：`npm run preview`

部署應用：構建後將 dist/ 文件夾內容部署到netlify 

連結：https://warm-sundae-283206.netlify.app/

# 概要

用於學習RWD響應式網頁設計

基於ReactJS 和 Tailwind CSS 建立和部署完整的旅遊網站

# 環境配置

開發環境：Windows 10　，　Visual Studio Code

# 專案結構

## 前端 

ReactJS、Tailwind CSS
